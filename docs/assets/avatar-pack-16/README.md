## 风格规范

这组头像基于当前产品英雄图的风格气质生成，统一特征包括：

- 浅色纸感背景
- 半写实、轻二次元人物绘制
- 温和未来感制服语言
- 圆形光晕背景
- 轻量点线、索引、图表、路径、文档等装饰元素
- 适合在 LLM 软件中作为角色 / 助手 / Persona 头像使用

## 使用建议

优先使用 `128x128/` 目录中的图片作为产品内头像。  
`original/` 目录适合用于大尺寸展示、角色详情页、二次裁切或后续设计加工。

建议在软件中保持统一圆角或圆形裁切，不要再叠加强边框，以免破坏原图的柔和纸感。
